# Global Retail Sales & Profitability Analysis

## Overview

This project explores a global retail dataset using Tableau to analyse sales performance, profitability, customer behaviour, and market trends.

The dashboard combines interactive visualisations to evaluate business performance across different markets, customer segments, product categories, and shipping operations.

## Objectives

The analysis aims to answer the following questions:

- Which markets generate the highest sales and profit?
- Which product categories perform best?
- How do customer segments contribute to revenue?
- What impact do shipping costs have on profitability?

## Dataset

The dataset contains over 51,000 retail transactions across multiple countries and markets, including information on customers, products, sales, profit, and shipping.
